三明治

笔记：

  • 用保鲜膜给三明治定型：将三明治放在保鲜膜正中，依次将四面翻折裹紧，静置 3 分钟后再切
  • 如果要切不止一刀，每切完一刀后用抹布擦干净刀上的酱汁，这样可以保持切面整齐
